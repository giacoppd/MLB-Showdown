Java Version of the Cardgame "MLB Showdown", Published 2000-2005 by Wizards of the Cost

This version allows for drafting of teams through either a text file or by querrying players currently added

Number of players added is about 6 teams worth from the 2004 Base Set

Pitcher functionality is lacking as presently the pitcher is just defaulted to Jarrod Washburn
